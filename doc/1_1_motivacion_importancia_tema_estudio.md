# 1.1 Motivación e importancia del tema de estudio

La accesibilidad web es un componente esencial para garantizar que todas las personas, independientemente de sus capacidades físicas o cognitivas, puedan interactuar con contenido digital de manera equitativa. Sin embargo, numerosos sitios web continúan presentando barreras que dificultan o impiden el acceso a usuarios con discapacidades. Detectar estos problemas suele requerir auditorías manuales complejas, costosas y poco escalables.  

La accesibilidad web es un derecho fundamental que garantiza el acceso equitativo a la información digital para todas las personas, incluidas aquellas con discapacidades visuales, auditivas, cognitivas o motoras. A pesar de la existencia de estándares internacionales como las Pautas de Accesibilidad para el Contenido Web (WCAG), una gran cantidad de sitios web continúa presentando barreras que dificultan la navegación y el uso por parte de estos grupos.
