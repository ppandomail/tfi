# 1.5 Objetivos del trabajo

## Objetivo General

Desarrollar un modelo de aprendizaje automático que clasifique páginas web según problemas de accesibilidad, para promover la inclusión digital y detectar  barreras que afectan a personas con discapacidad.

## Objetivos Específicos

* **Implementar un sistema automatizado de extracción de características** que permita analizar el contenido de páginas web para detectar las métricas de accesibilidad previamente definidas.

* **Entrenar un modelo de aprendizaje automático supervisado** para clasificar páginas web en función de la presencia o ausencia de problemas de accesibilidad, utilizando las métricas extraídas como variables de entrada.

* **Evaluar el desempeño del modelo propuesto** mediante métricas como precisión, recall, F1-score y matriz de confusión, para determinar su efectividad en la detección de problemas de accesibilidad.

* **Desarrollar una interfaz de usuario funcional** que permita ingresar una URL para obtener un diagnóstico automático sobre el nivel de accesibilidad de la página web analizada.
